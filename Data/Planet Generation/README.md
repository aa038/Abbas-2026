# Data/Planet Generation -- SAG13 Planet Catalog generation

This folder contains the star list and the scripts that generate the simulated planet catalog used throughout the paper.

## Files
- `HWO Stars.csv` - target star list (columns include HDName, `M [Msun]`, `L [Lsun]`, `Dist [pc]`, `SpecType`).
- `1. SAG13 Planet Population - Radius + Orb Params.py` - Part 1 (`Rp`, `P` and `orbital elements`).
- `2. Planet Radius to Mass and Albedo.py` - Part 2 (`Mass`, `Albedo`, `HZ flag`).
- `SAG13 Planet Catalog.csv` - **Final** planet catalog used by all figures after running both scripts.

## How to regenerate the catalog
From the repo root (or from this folder), run **in this order**:

```bash
python "Data/Planet Generation/1. SAG13 Planet Population - Radius + Orb Params.py"
python "Data/Planet Generation/2. Planet Radius to Mass and Albedo.py"
```

This will (re)create `SAG13 Planet Catalog.csv`.  
The second script **overwrites** the same CSV in place by adding `Mp_Mearth`, `albedo`, and `HZ`.
If you want two separate catalogs with and without mass, change the save path in `2. Planet Radius to Mass and Albedo.py`

## Determinism & re-runs
- Both scripts set `SEED = 42`. That makes the catalog **bit-for-bit reproducible** for the paper.
- If you delete or change the seed, the catalog **will change** (planet counts and properties).
- For paper reproduction, **do not edit** the CSV by hand and **do not** change the seeds.
- If you intentionally regenerate a new population for experiments, commit it separately and note the seed.

## Tunable ranges (Part 1)
- `Rp_range = [0.5, 3.4]` (Earth radii)
- `P_range  = [0.03, 10]` (years)

These define the SAG13 sampling box. All other behavior matches the paper.

## Methods (short)
- **Occurrence model:** broken power law in `Rp` with slopes/normalizations either side of `Rplim=3.4 R_E`, and a power law in `P`; planets per star drawn from Poisson with mean equal to the integrated rate.
- **Orbits:** Inclinations drawn from a `sin i` distribution; Rayleigh `e` (scale 0.061, clipped to [0, 0.95]).
- **Masses:** Chen & Kipping (2017) piecewise radius-mass with regime-dependent log scatter.
- **Albedos:** Simple empirical mass-class priors (rocky ~0.3; sub-Neptunes darker; giants brighter; BDs very dark).
- **HZ flag:** periastron/apastron both within nominal HZ (defined using Kopparappu 2013).
