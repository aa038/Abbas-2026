# Figure 1 -- Planet Radius vs Period

This folder reproduces **Figure 1** from Abbas et al. (2026, ApJ).  
The script plots all simulated SAG13 planets, highlighting exo-Earth candidates (**0.8-1.4 R_E within the HZ**).

## Files
- `Planet Radius vs Period.py` -- script to generate the plot.
- `fig1_Period_vs_Radius.png`  -- output figure (matches paper).

## Dependencies
- Python >3.9  
- `numpy`, `pandas`, `pathlib`, `matplotlib`
- The custom built plotting package `PlotStyle` (Optional). Installation instructions are in `REQUIREMENTS.md` in the root directory.
- Requires the processed planet catalog: `Data/Planet Generation/SAG13 Planet Catalog.csv`.
- That catalog is generated by the two scripts in the `Data/Planet Generation` directory:
  1. `1. SAG13 Planet Population - Radius + Orb Params.py`
  2. `2. Planet Radius to Mass and Albedo.py`
- If you want a new planet population, rerun **both** of these.

## Important note
- **You only need to regenerate the planet catalog if you want a new random realization.**   
- If you rerun the Data scripts without that seed, you will get a different planet population.

## How to run
From the repo root, run:

```bash
python "Fig 1 - Planet Radius vs Period/Planet Radius vs Period.py"
```

This will regenerate `fig1_Period_vs_Radius.png`, matching Fig 1 in Abbas et al. 2026
